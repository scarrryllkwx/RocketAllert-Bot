# Бот автопостинга MAX «RocketAlert»

Бот для мессенджера MAX: по нажатию кнопки публикует в канал сообщение
о ракетной опасности или об отбое. Доступ к управлению защищён паролем.

## Возможности

- Две кнопки: **«Оповещение»** и **«Отмена»**.
- Публикация готовых сообщений в канал одним нажатием.
- Авторизация по паролю (без пароля бот не управляется).
- Запоминание авторизованных пользователей между перезапусками.

## Установка

1. Установите Python 3.9+ и зависимости:
   ```
   py -3 -m pip install -r requirements.txt
   ```
2. Создайте файл настроек из шаблона и впишите токен и пароль:
   ```
   copy config.example.py config.py      (Windows)
   cp config.example.py config.py         (Linux)
   ```
   В `config.py` укажите:
   - `TOKEN` — токен бота MAX (из `@masterbot`);
   - `PASSWORD` — пароль для доступа к боту.
3. Добавьте бота **администратором** в нужный канал.

> Файлы `config.py` (секреты) и `authorized.json` (список пользователей)
> в репозиторий не попадают — они в `.gitignore`.

## Запуск

- Windows: двойной клик по **`Запустить бота.bat`** или `py -3 bot.py`.
- Linux/сервер: `python3 bot.py` (рекомендуется через systemd — см. ниже).

Пока процесс запущен — бот работает. Остановить: `Ctrl+C` или закрыть окно.

## Как пользоваться

1. Откройте бота **@id504602122857_4_bot** («Автопост») в MAX, нажмите `/start`.
2. Бот попросит **пароль** — введите его.
3. После авторизации появится **«Выберите действие:»** с кнопками:
   - **Оповещение** — публикует сообщение о ракетной опасности;
   - **Отмена** — публикует сообщение об отбое.
4. После публикации бот пишет **«✅ Сообщение опубликовано»** и снова
   показывает кнопки. Повторно вводить пароль не нужно.

## Важно

- Бот должен оставаться **администратором** канала
  (`https://max.ru/id504602122857_biz3`), иначе публиковать не сможет.
- Тексты сообщений находятся вверху файла `bot.py` (`TEXT_ALERT`, `TEXT_CANCEL`).
- Для круглосуточной работы держите бота на сервере (см. systemd-юнит в деплое).
